# 自定义时长功能测试指南

## 功能概述
为背诵板块的每个背诵项添加自定义时长功能，允许用户为不同的背诵内容设置不同的复习间隔。

## 测试步骤

### 1. 添加新背诵项并设置自定义时长
1. 打开应用并进入背诵板块
2. 点击"添加背诵内容"按钮
3. 输入背诵内容名称，如"英语单词"
4. 点击"自定义时长（可选）"按钮
5. 选择一个预设时长（如45分钟）
6. 点击"添加到树上"
7. 验证：新添加的项目应该显示所选时长

### 2. 编辑现有背诵项的时长
1. 在背诵列表中找到任意一项
2. 点击时钟图标 ✏️（或时长显示区域）
3. 在弹出窗口中选择新的时长
4. 点击"确定"
5. 验证：该项目的时长应该更新

### 3. 使用默认节奏
1. 在添加或编辑时长时
2. 选择"使用默认节奏"选项
3. 验证：该项目应该使用全局设置的节奏

### 4. 验证叶子变黄时间
1. 为某个项目设置自定义时长（如15分钟）
2. 完成一次背诵
3. 等待指定时间后
4. 验证：叶子应该按时长设置变黄，而不是全局节奏

## 预期行为
- 添加时长选择器时不应影响现有功能
- 自定义时长应优先于全局节奏
- 进度条应正确反映自定义时长
- 倒计时显示应准确
