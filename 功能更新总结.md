# 功能更新完成总结

## ✅ 已完成的功能

### 1. 背诵板块叶子一天只从黄变绿一次

**实现方式：**
- 叶子颜色状态只由 `lastMemorizedAt` 时间戳控制
- `completeRecite` 方法更新 `lastMemorizedAt` 为当前时间
- `refreshLeafColors` 每秒检查所有叶子，根据 `lastMemorizedAt` 和时长计算颜色
- 即使多次调用 `completeRecite`，也只会更新同一个时间戳

**关键代码：**
```typescript
completeRecite: (id) => {
  const now = Date.now()
  set((s) => ({
    reciteItems: s.reciteItems.map((r) =>
      r.id === id ? { ...r, lastMemorizedAt: now } : r
    ),
    // ...
  }))
}
```

### 2. 第二天叶子重置为0（黄色）

**实现方式：**
- 叶子颜色计算基于 `Date.now() - lastMemorizedAt`
- 第二天时间差超过24小时，必然大于任何时长设置
- 叶子自动变为黄色，等待复习

**关键代码：**
```typescript
const elapsed = now - item.lastMemorizedAt
const duration = item.customDecayDuration ?? DECAY_DURATIONS[state.decaySpeed]
const newColor = elapsed >= duration ? 'yellow' : 'green'
```

### 3. 背诵记录功能

**实现方式：**
- 新增 `ReciteRecord` 接口记录每次背诵
- `completeRecite` 时自动创建记录
- 记录包含：背诵项ID、名称、日期（YYYY-MM-DD格式）、时间戳

**数据模型：**
```typescript
export interface ReciteRecord {
  id: string
  reciteItemId: string
  itemName: string
  date: string  // YYYY-MM-DD
  timestamp: number
}
```

### 4. 月总结功能

**实现方式：**
- 新增 `MonthlySummary` 接口包含月度统计数据
- 新增 `DailyReciteStat` 接口记录每日背诵详情
- `getMonthlySummary(year, month)` 方法计算指定月份的统计
- 支持月份导航（上个月、下个月、回到本月）

**统计内容：**
- 当月总背诵次数
- 涉及的知识点数量

## 🔄 最近更新（2026-07-11）

### 新的苹果颜色系统

**绿色 → 红色 → 绿色 → 红色 循环**

**流程说明：**
1. 添加背诵项 → **绿色**（从未复习）
2. 自定义时长过后 → **红色**（可以复习）
3. 完成复习并选择感觉 → **绿色**（今天已复习）
4. 感觉持续时间过后或第二天 → **红色**（等待下次复习）
5. 重复步骤 3-4

### 感觉评级系统

完成背诵后可以选择复习效果：
- 😊 **掌握很好** → 绿色持续 24 小时
- 😐 **基本掌握** → 绿色持续 12 小时
- 😔 **还需加强** → 绿色持续 6 小时

### 自定义复习时长

- 每个背诵项可设置独立的复习时长（分钟）
- 添加时可选择设置
- 已添加的项可以随时编辑时长
- 不设置则使用全局默认时长

### 主要改进

1. **移除黄色状态**
   - 简化颜色系统，只有绿色和红色
   - 绿色含义：未复习 或 今天已复习
   - 红色含义：准备复习

2. **感觉驱动复习节奏**
   - 根据用户自我评估调整复习间隔
   - 掌握好 → 更长间隔
   - 需加强 → 更短间隔

3. **更流畅的复习流程**
   - 倒计时自动完成
   - 感觉选择器简洁明了
   - 颜色变化更加直观

4. **数据模型增强**
   - 添加 `customDuration` 字段
   - 添加 `feeling` 和 `lastFeelingAt` 字段
   - 完善复习记录系统
