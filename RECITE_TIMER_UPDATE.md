### 📝 更新内容

#### ReciteTimer 组件改为和专注模式一样的设计

**主要改进：**

1. **自定义时间改为分钟+秒**
   - 之前：只能输入分钟（1-999）
   - 现在：分钟 + 秒（类似专注模式）
   - 格式：`MM:SS`
   - 最小值：1分钟（60秒）

2. **新增暂停功能**
   - 添加暂停按钮
   - 暂停时显示"已暂停"
   - 可以继续计时

   - 完成后显示"再来一轮"按钮
   - 重新开始计时（不关闭面板）

4. **样式统一**
   - 使用 moss 颜色主题（和专注模式一致）
   - 相同的布局和间距
   - 相同的圆环进度条

### ✨ 用户流程

**完整使用流程：**

1. 点击红色苹果"开始背诵"
2. 选择时长（预设或自定义）
3. 点击"开始背诵"
4. **倒计时开始**（显示圆环进度 + MM:SS）
5. **随时可以暂停/继续**
6. 倒计时结束 → 显示"背诵完成"
7. 选择：
   - "再来一轮" → 返回设置界面重新开始
   - 点击关闭 → 返回列表
