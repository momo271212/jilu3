### 2. 颜色逻辑反转

#### 原来的逻辑（叶子）
```
背诵后：黄 → 绿
时间到：绿 → 黄
```

#### 新的逻辑（苹果）
```
背诵后：绿 → 黄
时间到：黄 → 绿
```

#### 含义变化
- **绿苹果** = 需要复习（时间到了）
- **黄苹果** = 已掌握（刚背完）
- **未背诵** = 绿苹果（初始状态）

## 🎯 设计理念

### 苹果的隐喻
- **绿苹果** = 青涩、需要学习
- **黄苹果** = 成熟、已掌握
- **从绿变黄** = 学习成长的过程

### 与"慢慢长"的结合
- 应用名称：慢慢长
- 苹果生长：从青涩到成熟
- 学习过程：从陌生到熟练
